### 소스코드의 구조
전체 코드를 모두 올린 것이 아니라 작성에 필요한 코드만 발췌하여 올린 것이다.<br/>
하위 디렉토리 단위에는 여러개의 프로젝트가 다수 존재한다.<br/>
보통 GitHub 자료를 받아서 바로 소스코드를 동작시키는 구조가 아니라 핵심소스만 발췌하여 올렸다.<br/>
사용법은 Anrdoid Studio 또는 VScode 에서 프로젝트를 생성할 때 <br/>
1) pubspec.yaml 파일의 첫줄 name 명이 프로젝트 명이므로 이 이름으로 프로젝트를 생성한다.<br/>
2) pubspec.yaml 파일에 추가된 라이브러리는 그대로 추가해주면 된다.<br/>
3) 생성된 프로젝트에서 assets 파일을 덮어쓰기 하고, lib 폴더도 덮어쓰기 하면 된다.<br/>
4) Android 또는 iOS 폴더에 있는 파일은 직접 내용을 살펴보고 생성된 프로젝트 부분과 다른 부분을 찾아서 추가하면 된다.<br/>

하지만 학습을 목표로 하는 것이므로 lib 폴더는 하나씩 참조하면서 구현해보는 것이 좋을 것이다.<br/>
